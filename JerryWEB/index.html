Om testet

Välkommen till Newtons webbportal för testfrågor.

Hos oss på Newton kan du som student göra tester och få reda på ditt svar direkt efter utförda test. Detta sker i form av flervals frågor och är tidsbestämt av din lärare.

Efter utförd kunskapskontroll/ test får du feedback utav din lärare om vad du eventuellt som kan förbättras inom det testade området.

Vid upplevda problem var vänlig kontakta administratören efter testet för feedback på vad om inte funkade.

Vi tackar för din medverkan och hoppas att allt har gått bra för dig i ditt test.
